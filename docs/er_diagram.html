1 ตั้งค่าแคตตาล็อกสินค้า

ผู้ดูแลร้านเตรียมข้อมูลสินค้าก่อนเปิดขาย — จัดหมวดหมู่ → เพิ่มสินค้า → แนบรูป

categories

สร้างหมวดหมู่สินค้า เช่น สมาร์ทโฟน, แล็ปท็อป

ออก PK id

→category_id
products

เพิ่มสินค้าเข้าหมวดหมู่ พร้อมราคาและรายละเอียด

FK category_id → categories

→product_id
product_images

แนบรูปสินค้าได้หลายรูปต่อ 1 สินค้า

FK product_id → products

2 ลูกค้าสั่งซื้อ

ลูกค้าเปิดคำสั่งซื้อ แล้วระบบบันทึกแต่ละรายการสินค้าลงในบิล

customers

ข้อมูลลูกค้า: ชื่อ ที่อยู่ เบอร์โทร

ออก PK id

→customer_id
orders

เปิดบิล 1 ใบต่อการสั่งซื้อ เก็บวันที่/สถานะ/ยอดรวม

FK customer_id → customers

→order_id
order_items

แตกบิลเป็นรายการสินค้า — สินค้าอะไร กี่ชิ้น ราคาต่อหน่วย

FK order_id → orders

→product_id
products

แต่ละรายการอ้างกลับไปยังสินค้าที่ขาย

FK product_id → products

💡 snapshot ราคา: order_items.price คัดลอกราคา ณ ตอนสั่งซื้อ แยกจาก products.price — ถ้าภายหลังปรับราคาสินค้า ยอดในบิลเก่าจะไม่เปลี่ยน

3 สถานะคำสั่งซื้อ (lifecycle)

ค่าในคอลัมน์ ENUM orders.status ไล่ไปตามขั้นตอนการจัดส่ง

processing → received → delivered
